# egg-born-module-cms-themecommunity

Cabloy-CMS Theme

## Installation

``` bash
$ npm i egg-born-module-cms-themecommunity
```

## Usage

Changing languange's theme to `cms-themecommunity`

``` javascript
"language": {
  "default": "en-us",
  "items": "en-us"
},
"themes": {
  "en-us": "cms-themecommunity"
}
```
